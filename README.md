# NEON SCAN
Network scanner built in Python. 
Designed to give a quick overview of open ports.

### Installation
~~~
git clone https://github.com/JohanGabrielson/neonscan.git
cd neonscan
pip install -r requirements.txt
~~~

### How to use
~~~
python3 main.py
~~~

### Project structure
~~~
neonscan/
│── main.py
│── scanner.py
│── network.py
│── output.py
│── README.md
~~~

### Intended use
Neon scan is intended to be use for education, CTF,  testing and legitimate security analysis. 
Do not use on networks without consent.
